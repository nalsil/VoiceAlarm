# Voice Alarm

보이스 알람은 사용자가 음성으로 알람 설정을 관리할 수 있는 안드로이드 애플리케이션입니다. 

## ✨ 주요 기능

*   **알람 추가, 수정, 삭제**: 간편하게 알람을 관리할 수 있습니다.
*   **요일별 설정**: 특정 요일에만 알람이 울리도록 설정할 수 있습니다.
*   **알람 활성화/비활성화**: 알람을 삭제하지 않고도 임시로 끄고 켤 수 있습니다.
*   **다국어 지원**: 알람 음성 언어를 한국어와 영어 중에서 선택할 수 있습니다.
*   **볼륨 조절**: 알람 소리의 크기를 조절할 수 있습니다.
*   **진동 설정**: 알람 시 진동 여부를 선택할 수 있습니다.

## 🛠️ 사용된 기술

*   **언어**: [Kotlin](https://kotlinlang.org/)
*   **UI**: [Jetpack Compose](https://developer.android.com/jetpack/compose)
*   **아키텍처**: MVVM (Model-View-ViewModel)
*   **데이터베이스**: [Room](https://developer.android.com/training/data-storage/room)
*   **비동기 처리**: [Coroutines](https://kotlinlang.org/docs/coroutines-overview.html)
*   [ViewModel](https://developer.android.com/topic/libraries/architecture/viewmodel)
*   [SplashScreen API](https://developer.android.com/develop/ui/views/launch/splash-screen)

## 🚀 시작하기

1.  이 저장소를 클론합니다.
    ```bash
    git clone https://github.com/nalsil/VoiceAlarm.git
    ```
2.  Android Studio에서 프로젝트를 엽니다.
3.  필요한 Gradle 종속성이 동기화될 때까지 기다립니다.
4.  애플리케이션을 빌드하고 실행합니다.
5.  개인정보처리지침 URL
    ```
    https://nalsil.github.io/VoiceAlarm/doc/privacy_policy
    ```
